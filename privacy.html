プライバシーポリシー

最終更新日: 2026年3月25日

Richend Digital（以下「当社」）は、予約管理サービス「カンタン予約」（以下「本サービス」）における、ユーザーの個人情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

第1条（個人情報の定義）

本ポリシーにおいて「個人情報」とは、個人情報保護法に定める「個人情報」を指し、生存する個人に関する情報であって、当該情報に含まれる氏名、メールアドレス、その他の記述等により特定の個人を識別できるもの、または個人識別符号が含まれるものを指します。

第2条（収集する情報）

当社は、本サービスの提供にあたり、以下の情報を収集することがあります。

  1. アカウント情報: 氏名、メールアドレス、店舗名、電話番号
  2. 決済情報: クレジットカード番号（決済代行会社を通じて処理され、当社では保持しません）
  3. 利用情報: サービスの利用状況、ログ情報、アクセス日時
  4. 端末情報: IPアドレス、ブラウザの種類、OS情報
  5. 予約データ: ユーザーが本サービスを通じて管理する予約情報・顧客情報

第3条（利用目的）

当社は、収集した個人情報を以下の目的で利用します。

  1. 本サービスの提供・運営・改善
  2. ユーザーからのお問い合わせへの対応
  3. 利用料金の請求・決済処理
  4. サービスに関する重要なお知らせ、メンテナンス情報の通知
  5. 新機能・キャンペーン等のご案内（ユーザーが同意した場合）
  6. 利用状況の統計・分析（個人を特定しない形式）
  7. 不正利用の防止・セキュリティの確保
  8. 利用規約に違反する行為への対応

第4条（第三者提供）

  1. 当社は、以下の場合を除き、あらかじめユーザーの同意を得ることなく、第三者に個人情報を提供することはありません。
    • 法令に基づく場合
    • 人の生命・身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
    • 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
    • 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合
  2. 当社は、本サービスの提供に必要な範囲で、以下の第三者にデータ処理を委託することがあります。
    • クラウドインフラ提供事業者（Google Cloud / Firebase）
    • 決済代行事業者
    • メール配信サービス事業者

第5条（安全管理措置）

当社は、個人情報の漏洩、滅失またはき損の防止その他の個人情報の安全管理のために、以下の措置を講じます。

  • SSL/TLSによる通信の暗号化
  • アクセス権限の適切な管理
  • データベースの暗号化
  • 定期的なセキュリティ監査
  • 従業者に対する個人情報保護に関する教育

第6条（ユーザーの権利）

ユーザーは、当社に対して以下の権利を行使することができます。

  1. 開示請求: 当社が保有する個人情報の開示を請求できます
  2. 訂正・削除: 個人情報の内容が事実でない場合、訂正または削除を請求できます
  3. 利用停止: 個人情報の利用停止を請求できます
  4. データポータビリティ: ご自身のデータを構造化された形式で受け取ることを請求できます

上記の請求は、お問い合わせページよりご連絡ください。ご本人確認の上、合理的な期間内に対応いたします。

第7条（Cookieの使用）

  1. 本サービスでは、ユーザー体験の向上およびサービスの改善のためにCookieを使用しています。
  2. Cookieとは、ウェブサイトがユーザーのブラウザに送信する小さなデータファイルです。
  3. 当社が使用するCookieの種類:
    • 必須Cookie: サービスの基本機能（ログイン状態の維持等）に必要
    • 分析Cookie: サービスの利用状況の把握・改善に使用（Google Analytics等）
  4. ユーザーは、ブラウザの設定によりCookieの受け入れを拒否することができます。ただし、一部の機能が正常に動作しなくなる場合があります。

第8条（EU域内ユーザーへの対応）

EU一般データ保護規則（GDPR）が適用されるユーザーに対しては、以下の追加的な権利を保障します。

  • データ処理に対する同意の撤回
  • 自動処理のみによる意思決定の制限
  • 監督機関への苦情申立て

第9条（本ポリシーの改定）

  1. 当社は、必要に応じて本ポリシーを改定することがあります。
  2. 重要な変更がある場合は、本サービス上での通知またはメールにてお知らせします。
  3. 改定後のプライバシーポリシーは、本ページに掲載した時点より効力を生じるものとします。
